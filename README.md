## Descripción
Este repositorio contiene la integración solicitada con la pasarela de pagos Khipu, utilizando su API REST para generar un cobro dinámico de $5.000 CLP y procesarlo mediante el entorno de pruebas DemoBank.

## Notas sobre la Integración

El script `main.py` adjunto fue construido bajo esta arquitectura oficial, logrando la conexión exitosa.
